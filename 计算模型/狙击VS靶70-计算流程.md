# 狙击 VS 靶70 计算流程

本文档说明的是**当前项目实际代码**对场景“狙击 VS 靶70”的计算过程，不是按 RimWorld 原版机制重新手算的一套独立模型。

## 1. 场景输入

- 场景名：`狙击 VS 靶70`
- 攻击方：`狙击`
- 防守方：`靶70`
- 距离：`40 格`
- 场景命中率倍率：`100% = 1.0`
- 掩体阻挡：`0`
- 攻击方射击等级：`14`
- 攻击方特性/增益：`无`
- 武器：`大师级 狙击步枪/vanilla`
- 防守方护甲：`全身 70%`

## 2. 当前项目读取到的武器数据

当前项目里，这把武器在计算时使用的是：

- 基础伤害：`25`
- 大师级伤害倍率：`1.08`
- 最终单发伤害：`25 x 1.08 = 27.00`
- 基础长距离命中系数：`0.90`
- 大师级精度倍率：`1.08`
- 最终长距离命中系数：`0.90 x 1.08 = 0.9720`
- 基础瞄准时间：`3.50s`
- 大师级周期倍率：`0.96`
- 最终瞄准时间：`3.50 x 0.96 = 3.36s`
- 基础冷却时间：`1.50s`
- 最终冷却时间：`1.50 x 0.96 = 1.44s`
- Burst 数：`1`
- 武器护甲穿透：`0.00`

## 3. 14 级射击在 40 格的命中率（先不算武器）

当前项目对远程射击者先算“每格精度”：

- 射击等级：`14`
- 无特性修正
- 视觉：`1.0`
- 操作：`1.0`
- 因此射手精度统计值仍为：`14`
- 查表得到每格精度：`0.98`

只看人物，不乘武器时，40 格距离衰减为：

`0.98 ^ 40 = 0.445700403951`

也就是：

- **14 级射击在 40 格、先不算武器时的命中保留率：`44.57%`**

## 4. 最终命中率

40 格在当前项目里属于 `long` 距离段，因此乘武器长距离命中系数 `0.9720`。

计算过程：

`最终命中率 = 人物距离衰减 x 武器长距离命中系数 x 场景命中率倍率 x (1 - 掩体阻挡)`

代入：

`最终命中率 = 0.445700403951 x 0.9720 x 1.0 x (1 - 0)`

结果：

`最终命中率 = 0.433220792640`

也就是：

- **最终命中率：`43.3221%`**

## 5. 射击周期

当前场景下没有额外瞄准时间和冷却时间修正：

- 瞄准倍率：`1.0`
- 冷却倍率：`1.0`

因此：

- 实际瞄准时间：`3.36 x 1.0 = 3.36s`
- 实际冷却时间：`1.44 x 1.0 = 1.44s`
- Burst 追加间隔：`0s`

所以单轮射击周期：

`射击周期 = 3.36 + 1.44 + 0 = 4.80s`

也就是：

- **射击周期：`4.80 秒`**

## 6. 穿甲与护甲结算

### 6.1 当前项目中的有效穿甲值

- 武器护甲穿透：`0.00`
- 攻击方没有额外护甲穿透倍率
- 所以有效穿甲值：`0.00`

### 6.2 当前项目中的有效护甲

- 防守方全身护甲：`70`
- 有效护甲：`70 - 0 = 70`

### 6.3 当前项目如何把 70 护甲转成伤害结果

当前项目把有效护甲 `70` 转成三段概率：

- 弹开概率：`70 / 200 = 0.35`
- 全伤通过概率：`1 - 70 / 100 = 0.30`
- 半伤概率：`1 - 0.35 - 0.30 = 0.35`

所以单发命中后的结果是：

- `35%` 完全弹开
- `35%` 只造成半伤
- `30%` 造成全伤

## 7. 穿甲效率

“穿甲效率”这个名字在当前项目里没有单独字段。为了方便核对，这里给出两个值：

### 7.1 有效穿甲值

- **有效穿甲值：`0.00`**

### 7.2 对 70% 护甲目标的平均伤害保留率

这个值更接近“这发子弹平均能穿过去多少”：

`平均伤害保留率 = 0 x 0.35 + 0.5 x 0.35 + 1.0 x 0.30 = 0.475`

也就是：

- **平均伤害保留率：`47.50%`**
- **平均护甲减伤率：`52.50%`**

如果你要把它记成“穿甲效率”，那当前这一枪打 70 护甲靶时：

- **穿甲效率（按平均伤害保留率理解）：`47.50%`**

## 8. 命中后期望伤害

单发原始伤害为 `27.00`。

命中后的期望伤害：

`27.00 x 0.475 = 12.825`

也就是：

- **命中后期望伤害：`12.8250`**

## 9. DPS

### 9.1 理论 DPS

`理论 DPS = 单发原始伤害 / 射击周期`

`理论 DPS = 27.00 / 4.80 = 5.625`

### 9.2 期望 DPS

`单轮期望伤害 = 最终命中率 x 命中后期望伤害`

`单轮期望伤害 = 0.433220792640 x 12.825 = 5.556056665612`

`期望 DPS = 单轮期望伤害 / 射击周期`

`期望 DPS = 5.556056665612 / 4.80 = 1.157511805336`

最终：

- 理论 DPS：`5.6250`
- 期望 DPS：`1.1575`

## 10. 本文档中的关键结论

- 14 级射击在 40 格、先不算武器时的命中保留率：`44.57%`
- 最终命中率：`43.3221%`
- 射击周期：`4.80 秒`
- 有效穿甲值：`0.00`
- 对 70 护甲的平均伤害保留率：`47.50%`
- 命中后期望伤害：`12.8250`
- 理论 DPS：`5.6250`
- 期望 DPS：`1.1575`
